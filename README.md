# ATmega328P PCB Design (Beginner Project)

Tool Used: EasyEDA

## Overview
This is a beginner-level PCB design project created as part of my learning
journey in PCB design and embedded systems. The board is based on the
ATmega328P microcontroller.

## Work Done
- Created schematic using ATmega328P and basic supporting components
- Designed PCB layout with proper component placement
- Performed basic routing
- Verified the design using EasyEDA 3D view

## Learning Outcomes
- Schematic-to-PCB workflow
- Basics of PCB placement and routing
- Introduction to ATmega328P microcontroller

## Project Status
Learning / practice project
